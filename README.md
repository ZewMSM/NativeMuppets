# ZewMuppets Native

Today I figured out native programming for myself, and as a showcase of the work I built a fully native mod for My Muppet Show.Today I figured out native programming for myself, and as a showcase of the work I built a fully native mod for My Muppet Show. This mod lets you play the game with no servers. All the game logic is baked straight into the engine, so it doesn't need to run a server in the background. Because of that, the game loads 5x faster. It also runs more stable across all devices.

**Meet the native experience.**:
- Fully offline
- No extra apps
- Free, no ads
- Mod it freely
- Clean, no watermarks

**Additional Patches**:
- Game loads 5x faster
- God mode and Reset progress in settings
- Full Zoom enabled by default
- Remove Age checks and review popups

Minimal API set to 21, Target API set to 23
